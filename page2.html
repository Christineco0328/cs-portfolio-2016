How to Code a Webpage with Javascript

Here I will use my own Calculator to show you how to use Javascript to make an interactive Calculator!

Creating the Javascript for your Calculator

Step 1: Variables

Variables are a way of storing information. Once we set something equal to the created variable, we can simply use the varible whenever we need to refer or include the information that we set it equal to. The main purpose of doing this is so that we do not have to write out the entire piece of data each time we need it in our code.
Setting your variable looks like this:

var *insert variable name* =

Naming your Variable

When naming your variable, make it something relevant. Just by looking at the variable, you should be able to tell which piece of information the variable contains. In addition, you don't want to make it overly complicated given that the main purpose of using a variable is to make your code simplier.

Step 4: If statements

If statements allow us to write functions that the program will carry out based on whether a certain statement is true or false. In the terms of our calculator, we will use an if statement to tell the program to execute our first operation.

We will begin writting within the curly brackets of the addEventListener we just created.

Format for Condition

First, we will type "if" to indicate to the program that we are writing an if statement. Then, within parentheses is where we will write our condition. Insert the name of the variable that is set to *the name of the variable linked to our select element*.value (the setting of these variables was established in the addEventListener step). Once you have this name follow it with three equal signs and the operation in quotations marks. In javascript, the significance of the three equal signs is used to check if both sides of the equal signs are in fact equal. If true, the program will proceed into carrying out the codeblock.

Writing the Codeblock

Codeblock is the code that the program will run if the if statement is true. We will begin writing our codeblock within curly brackets. To carry out operations, we are going to need three different variables:

1. Create a new variable and set it to parseInt followed by (name of varible that is set to the first input element value) established when we created our addEventListener. Don't forget the parentheses!

The first part of our new variable, parseInt is the key factor in allowing the calculator to read what is put into the input element. When the user inserts a number into the element, the program naturally reads it as a string rather than a number. In javascript, there are three different types of data that variables can be set to: strings, numbers, and boolean. Strings hold information such as characters, letter, and symbols, all of which that are uncomprehensive to a calculator. Thus, we have to make sure that whatever is typed into the input element is stored as a number--another type of data that stores numbers for their numerical value rather than a character. ParseInt is what makes this possible. It takes the string that is input when the user writes in the input element, and converts it to a number--allowing the calculator to operate on the number that is input.

Then, we also then have to clarify to the program what it is taking the parseInt of. Thus, we must put the name of the var set to *input element var name*.value in parentheses.
