How to Code a Webpage with Javascript

Here I will use my own Calculator to show you how to use Javascript to make an interactive Calculator!

Creating the HTML for your Calculator

Step 1: Inputs

In order to create a calculator, we are going to need an element that will allow us to type in our different factors. Inputs do just the job for us!

Input Tag

When creating an input, in our HTML we will use the tag:

<input>

One key difference between input tags and other tags such as headers, tables, or divs, is that it is self closing. This means that it will only need the initial tag that we use to start most tags. For our calculator, we are going to need to create two inputs for the two different factors that we to either add, subtract, multiply, or etc. together.

Step 2: Dropdowns

To give our calculator the capability of being able to carry out different functions, we can use the select and option tags to list out the different operations the calculator can do. Your tag should look like this:

<select></select>

Within your select tag you are going to nest each of your option tags where you will put in each of the different operations. Your option tag should look like this:

<option></option>
