Array Index:

The different elements within an array are given a number index so that we can refer to it later. The first element starts at 0 and numbers off by 1 for each of the following elements in that array.

For Loop Syntax:

There are four different parts to a for loop:

1. Insert the key word "for" to indicate that you are creating a for loop

2. Statement 1 will indicate at which index the codeblock should start running through the array.

3. Statement 2 is our condition statement. The codeblock will only run if this statement is true.

4. Statement 3 tells our program how often it should loop. An example of this would be every index value or every other index value, which would be expressed using the operator ++ (increment) or -- (decrement).

These three statements will be enclosed in parentheses and separated with semicolons.
