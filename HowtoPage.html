How to Code a Webpage using HTML and CSS

Here I will be using my own hardware page to show you how to code and create a webpage!

Headers

Step 1: HTML for your Headers

Headers are a big part of organization. They will allow you to tell your viewer what each new section is about. Through html, headers will naturally stand out more compared to regular text.


Header Tag

When you type into your html file you will start your header with:

<h1> </h1>

Your code should look like this.

Header 1 will show up as the biggest and most domaniant header. Further along, when you want to head the individual sections of your webpage, you can number your header as 2 and up. Lower-numbered headers will be be smaller and will not stand out as much. For my Hardware page specifcally, I only used "h2" to head the different categories so that I could keep the size and styling of the headers consistent.

Step 2: CSS for your Headers

CSS for your Headers will allow you to decorate your headers with borders, color, and fonts, while also allowing you to position where they go.

Centering your Headers

At this point, all of our headers are flushed all the way to the left. However, I want to center all of my headers so that the focus is in the middle of the page. To do this I used the property of "text-align" and set the value as "center." It should be formatted like this under your style tag:

text-align: center;

Borders

Since I have two different headers of different importance, I want to incorporate another design element that will draw more attention to my h1 header at the top of the page. I can do this through different styled borders.

Thus, in my style tag, I coded a solid border under my h1 selector, and a dotted border for all of my h2 tags, using the declarations:

border: 2px solid gray;

border: 8px dotted khaki;

(the unbolded information are design elements for the border width and color)

This is what it should look like!

Tables

Step 3: HTML for your Tables

Tables, when making a website, are a sufficent selector used to organize your information and group text together. For instance, throughout your page you can use tables to separate different topics. However, one disadvantage of tables is that certain common commands will not work with them. In general, they are more limiting in terms of designing a website. A better alternative would to use a selector called "divs," (short for divisions). Divs are still capable of achieving the same look tables can create, yet it will allow you to incorporate more design elements. On the other hand, I started my website unsure of what a "div" was so I decided to use tables instead, which still gets the job done.

HTML for Tables

<table> </table>

This tag will mark the start and end of the table

<tr> </tr>

"tr" will indicate where a table row starts. Add another table row when you want to make another row below.

<td> </td>

Finally, The tag "td" is where you will insert all of your table data. When you add more "td" tags in your table row, you will create more columns.

This is what it should look like!

Step 4: CSS for your tables

Background Color

Since I have a set background image for my webpage, text will not show up as clearly when only placed on top of the set background. To create more contrast between the words and the background, I decided to color in the background of the table so that the text would stand out more.

This can be done by going back to the style tag and setting the background color for the tables specifically. This element will be formatted like this:

background: white;

(any other color can be switched our for white)

Borders

To make my tables stand out more, I decided to create a border for all of my tables. This can be done the same way you can add borders to your headers. However, you must add the phrase "border: 2px solid gray;" to "table" under your style tag.

This is what it should look like!

Images

Step 5: HTML for Images

Images are important in a webpage. They allow the viewer to better understand the topic by giving them a visual to associate the text with.

When adding in images to your code, you can either upload them from your own files or from the internet. For my hardware page in particular, I only got images from the internet.

Thus, to insert images from the internet, you would use the tag:

<img src="*insert url here*/>

This is what it should look like!

Step 6: CSS for Images

There are many styling elements that can be applied to images, some of which include adding borders, changing the height, or width.

Height OR Width

Often times when I get an image from the internet, the size of the image is too large or too small to fit in my table. To solve this problem, CSS allows me to set the height or the width of the picture.

To set the height and the width, you can nest a style tag within your picture tag. After your picture url, you will write:

<img src="*insert url here* style="height:___px" />

<img src="*insert url here* style="width:___px" />

Usually you will only set one or the other, because once you set one, it will dialate the picture proportionally.

Borders

As for tables and headers, the CSS tag for headers will remain the same; however, it will be written under the image tag under CSS.

This is what it should look like!
