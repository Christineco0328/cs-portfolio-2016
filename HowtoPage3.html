How to Create Clickable Keyboard

Here I will show you how use Javascript, HTML, and CSS to create a Clickable Keyboard!

HTML for your Keyboard

Step 1: Create your Keys (Divs! Divs! Divs!)

In order to create a piano, we need to have keys to press. Therefore, in our HTML we can use divs to represent the keys on the keyboard. You will need to create a new div for each key you make, thus I suggest that you at least create 8 divs to complete a full octave on the piano.

Step 2: Give each Key an ID

In our javascript we will need to refer each key individually to assign specific sounds to each key according to their place on the keyboard. Thus, to make each key unique, we can give each div a different ID.

As mentioned in previous How to Pages , adding an ID will look like this:

<div id = "C" > </div>

Javascript for your Keyboard

Step 1: Get Element By Id

Through javascript, we will attach the different pitches found in the scale to each of the divs (keys). In order for us to interact with and manipulate the divs, we need to create a connection between the divs in our HTML and our javascript. We can achieve this using the method getElementByID().

This method takes the element in the HTML based on its ID and brings over its information onto the Javascript so that we can manipulate its data. Therefore, to bring over the data associated with the keys in our HTML, we will insert the ID of each of our divs into the getElementByID method.

Step 2: Add Event Listeners

Next, we want the program to be able to play a certain note when a key on our physical keyboard is pressed. In order for this to happen, we need our program has to know when the key is being pressed. Thus, we can use an Add Event Listener to enable our program to listen for when a key is pressed.

First, we need to tell the program which element it should be listening to. The program needs to be listening whenever the user is on the webpage. The element document refers to the entire HTML document. Thus, we can attach the add event listener to the page as a whole so that the add event listener is listening whenever the document is being used. Your code should now look like this:

document.addEventListener

Getting more specific, we now need to tell the program WHAT it needs to listen for. Using the event "keydown", the program will listen for when a key on the physical keyboard is pressed.
